Class Logger

description:

You can use this class in 2 variants: tagged and untagged, tagged: uses tagged template and reduce the ease of use for the dev. untagged: The dev is responsible of how to pass a correct string into the function.

example

tagged: logger.debugMy message is ${{type: "human", name: "Plony", lastName: "Almony"}};

example

untagged: logger.debug(Helo world);
